Abstract

A unified loss for document image understanding

CHULA is a novel Custom Heuristic Uncertainty-guided Loss designed for highly accurate segmentation and detection of Thai land title deed documents. It uniquely combines class-balanced cross-entropy, aleatoric uncertainty modeling, and domain-specific heuristic priors into a single differentiable objective.

The loss integrates seamlessly with any modern backbone — including YOLOv8, v11, v12, and DeepLabv3+ — delivering consistent performance gains on noisy, ambiguous, and structure-rich document imagery, particularly for underrepresented categories such as PIN markers and official seals (STAMP).

Key Contributions

What CHULA introduces

01

Unified Loss Function

A single objective combining uncertainty, class balance, and document heuristics — no auxiliary tricks or multi-stage training required.

02

Low-resource Class Recovery

Engineered to regain precision on rare semantic classes like PIN markers and seals, where standard cross-entropy catastrophically under-performs.

03

Plug-and-play Integration

Drop-in replacement for standard loss in YOLO, DeepLabv3+, and beyond. Two lines of Python — no architectural changes needed.

04

Multi-task Learning Support

Native support for joint segmentation and detection objectives in a single training run, enabling richer document representation learning.

Methodology

The CHULA loss formulation

Three complementary terms, each targeting a distinct failure mode in document image segmentation, combined into a single principled objective.

ℒCHULA  =  λce · ℒCE  +  λunc · ℒUNC  +  λheu · ℒHEU
ℒCE — Class-balanced cross-entropy  (λ = 1.0)
ℒUNC — Aleatoric uncertainty penalty  (λ = 0.3)
ℒHEU — Domain heuristic prior  (λ = 0.5)

Implementation

Quick integration

Designed for frictionless adoption into any YOLO or segmentation training loop.

chula_integration.py
from chula.loss import CHULALoss from chula.utils import compute_class_weights # Auto-detect binary vs multi-class & compute weights class_weights = compute_class_weights( "datasets/medical-pills", num_classes=1 ).cuda() # Instantiate with tunable component weights loss_fn = CHULALoss( class_weights=class_weights, lambda_ce=1.0, lambda_unc=0.3, lambda_heu=0.5 ) # Drop-in replacement for standard criterion loss = loss_fn(pred_logits, targets, uncertainty_map)
